Ü1. Eine Datei schreiben

$ vi

i (insert)

Hello

<Esc>

:w hello.txt (write)

:q (quit)


Ü2. Verschiedene Versionen von "insert"

vi hello.txt

I My🗻<Esc>

a dear🗻<Esc>

Hinweis: u (undo) [C-r (redo) später]

A 🗻world<Esc>

("My dear Hello world" speichern)


Ü3. Neue Zeile

$ vi hello.txt

O Tschierv 4. Juni 2011<Esc>

o Liebe Geeks,<Esc>

h j k l (Pfeile!)


Ü5. Schnelle Bewegungen: Homeline 

fx (forward to x)

tx (to x)

w (word - nicht alphanumerisch)

W (word - Leerschlag getrennt)

b (word back - nicht alphanumerisch)

B (word back - Leerschlag getrennt)

e (end of word - nitht alphanumerisch)

E (end of word - Leerschlag getrennt)

nw, nW

0 (Zeilenanfang)

$ (Zeilenende)


Ü6. Schnelle Bewegungen, über Zeilen hinaus

C-F (page forward)

C-B (page backward)

nG (jump to line n)

G (letzte Zeile)

gg od 1G (erste Zeile)

/ (forward search)

? (backward search)

n (repeat previous search)

N (repeat previous search backwards)


Ü7. Cut-and-Paste

<BS> nur im Insertmodus!

x (del under cursor)

X (del left of cursor)

dw, dW, db, dB, etc (delete movement)

ndw, ndW, etc. (n times delte movement)

dd (delete line)

ndd (delete n lines)

p (after cursor)

P (paste before cursor)

Hinweis: . (repeat last!)

Tests

  1. Zwei Zeichen tauschen.
  2. Zwei Wörter tauschen.
  3. Zwei Zeilen tauschen.


Ü8. Suchen, Ersetzen

:s/foo/bar/

:s (repeat last s)

r x (replace char with x)

nr x (replace next n char with x)

R text<Esc> (replace until Esc)


Ü9. Zur Motivation

:r file

:! shell command

:r ! shell command
